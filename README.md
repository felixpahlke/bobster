# Bobster

Bobster is a CLI and public registry for reusable Bob assets.

Bob is IBM's coding agent. Bobster will help teams discover, install, update, and share Bob skills, rules, and custom modes.

This package name is reserved while the first implementation is in progress.

## Planned Usage

```sh
bobster init
bobster search frontend
bobster add frontend-design
```

## Status

Initial implementation is not released yet.
